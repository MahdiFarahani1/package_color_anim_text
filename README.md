# color_anim_text

## Features

This package creates a very beautiful animation on your text, where a special color passes over your text or widget.

## Getting started
       WidgetAnimColor(
                color: Colors.black,
                highlightColor: Colors.red,
                child: Text("Hi Guys"),
              ),
